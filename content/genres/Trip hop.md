# Trip hop

Trip hop is a fusion music genre that originated in the UK in the 1990s.

|              |                                  |
| ------------ | -------------------------------- |
| AKA          | Downtempo                        |
| Origins      | 1990s UK                         |
| Subgenre(s)  |                                  |

## About
Trip hop pulls from a variety of genres to produce an atmospheric, psychedelic soup of slow-motion sound.
